# MoodApp

Share your Mood on the Blockchain. Basic dApp built on Ethereum with Solidity and JavaScript!

**About this Repository**<br />
This project is part of the "_[Ethereum Developer Degree - Freshman track](https://learnweb3.io/degrees/ethereum-developer-degree/freshman/)_" from [LearnWeb3](https://learnweb3.io/).<br />
It's a simple tutorial on how to create and deploy a Solidity smart contract and interact with it from a frontend website.
